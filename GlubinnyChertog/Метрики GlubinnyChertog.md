Проверить core loop игры на реальном трафике до вложения полного цикла разработки (14-16 недель).

## 3. Решение Go / No-Go

**🔴 Стоп / переделать core loop:**
- D1 < 25% — игра не цепляет с первого захода
- Действие: пересмотреть первые 60 секунд игры, темп Intro-фазы, ощущение от боя

**🟡 Доработать перед полным MVP:**
- D1 25–35%, CPI в пределах нормы
- Действие: точечно улучшить прогрессию, баланс волн, ревайв-механику; повторный тест через 1-2 недели

**🟢 Зелёный свет — продолжать полный MVP:**
- D1 35%+ и/или D7 ROAS 15%+
- Действие: переходить к полному циклу разработки, арту, контенту
